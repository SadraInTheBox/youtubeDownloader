Youtube downloader
==================

- develope by Z3R0 - Sadra

how to run
----------

`python main.py`

`[i]youtube url > {your youtube video url}`
exapmle : `[i]youtube url > https://www.youtube.com/watch?v=w_XPqOxHrYI`

```
1 - 1080
2 - 240
3 - 750
```

`select resolution > {resolution number}`
example : `select resolution > 2`
